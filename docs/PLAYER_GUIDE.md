# BUGSMASHER — Player Guide

## Objective

Defend the **Core** at the center of the screen. Eliminate bio-luminescent bugs before they drain your system health.

## Controls

| Input | Action |
|-------|--------|
| Click / Tap | Smash bugs (within cursor radius) |
| Hover | Collect hover-type powerups |
| Space / Shift | Dash toward cursor (cooldown) |
| Gamepad stick | Move aim (Settings → Gamepad Support) |
| Gamepad A / RT | Fire / smash |

## Core Loop

1. **Wave starts** — bugs spawn from screen edges  
2. **Prioritize** — scouts (fast), tanks (armored), healers (support)  
3. **Collect** — powerups and resource drops for meta upgrades  
4. **Survive surges** — intensity spikes between waves  
5. **Boss waves** — every 10 waves (configurable in daily challenge)

## Meta Progression (Main Menu)

- **Upgrades** — click radius, turrets, core health  
- **Progression Center** — craft resources, skill tree  
- **Armory** — cursor skins and core themes  
- **Daily Challenge** — unique modifiers + cosmetic rewards  

## Settings → Accessibility

- **Difficulty** — Easy / Normal / Hard (enemy speed, HP, drops)  
- **Reduced Motion** — disables screen shake  
- **Enemy Shape Icons** — shape markers on bugs (colorblind assist)  
- **Colorblind Assist** — applies display filter to game canvas  
- **Gamepad Support** — enable controller aiming  

## Tips

- Overheating: rapid clicks build weapon heat — wait for cooldown  
- Dash through clusters to reposition  
- Disable High Fidelity VFX on low-end devices (Settings → Visuals)
